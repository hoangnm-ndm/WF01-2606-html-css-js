# Phân loại kiến thức

## Kiến thức nền tảng

- Khai báo biến: var (bỏ qua - không học kỹ), let, const.
- Kiểu dữ liệu:
  - Primitive type: number, string, boolean, null, undefined, symbol.
  - Reference typy: object, array, function, Date, Map, Set.
- Toán tử:
  - Toán tử số học: +, -, \*, /, %, ++, --.
  - Toán tử so sánh: ==, ===, !=, !==, >, <, >=, <=.
  - Toán tử logic: &&, ||, !.
  - Toán tử gán: =, +=, -=, \*=, /=, %=.
  - Toán tử chuỗi: + (nối chuỗi).
- Các cấu trúc điều khiển:
  - Câu điều kiện: if...else, switch...case.
  - Vòng lặp: for, while, do...while.
- Hàm: function, arrow function.

## Các kiến thức nâng cao

- Thao tác DOM
  - Sự kiện
  - Form
- Bất đồng bộ
  - Promise
  - Async/Await
  - Fetch API
  - Callback
